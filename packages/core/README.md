# @0xagentio/core

Core TypeScript primitives for AgentIO.

This package contains dependency-light protocol types and helpers: identities, credentials, policies, validation, commitments, delegation statements, proof adapter contracts, storage/transport/execution contracts, and the Noir authorization input bridge.

Most application developers can import these through `@0xagentio/sdk`. Use `@0xagentio/core` directly when building adapters or lower-level integrations.
